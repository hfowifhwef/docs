Introduction to MessageBackupBot

The MessageBackupBot, is the all new discord bot, allowing guild messages to be backed up, and restored. In this documentary, it shows you on how to set it up, and how to use it.

Installation

First, to start using the bot, you need to invite it, click the link here.

Then, you want to select the “invite” link, and press “Invite To Your Server”. Please make sure you give it administrator, or, at least:

  • Webhooks
  • Emojis
  • Send messages
  • Read messages
  • Manage server
  • Manage channels
  • Embed
  • Upload files
  • Images

Without these, the bot will fail to work as expected.

Limitations

The Message Backup Bot, has some limitations.

  • You can save 6 channels worth up too 400 Messages (Free) [If you want more, please, view the “Upgrades” section of the wiki]
  • Restoring takes a while
  • The “bot” shows next to your name.

A guide too exporting your backups

It's 29 days past, your running out of time? How do I save my backups forever? With the all new exporting feature of course!

Follow this exact guide to learn how to export.

  1. Have your backup created.
  2. Have your encryption key and backup id saves or written somewhere.
  3. Type m#export [backup id]then, go to your dms, and enter your encryption key, after a while, return too your guild, and you will see this:

4. Please the blue “here” button and save this file safe.

5. ALSO SAVE YOUR EXPORT ID, THIS IS VITAL!

6. At any-time, you can drag and drop your text file into a discord server with the message m#import [export id]with the attachment (obviously a guild with MessageBackupBot in it) and you will then have your content uploaded too our servers.
