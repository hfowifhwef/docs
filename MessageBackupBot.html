Introduction to MessageBackupBot

The MessageBackupBot, is the all new discord bot, allowing guild messages to be backed up and restored. In this documentary, we will show you on how to set it up and how to use it.

Installation

To start using the bot, you need to invite it by clicking here.

Then, you want to select the “invite” link, and press “Invite To Your Server”. Please make sure you give it administrator or at least these permissions:

  • Manage Webhooks
  • Manage Emojis and Stickers
  • Send Messages
  • Read Messages
  • Manage Server
  • Manage Channels
  • Embed Links
  • Attach Files
  • Slash Commands

Without these permissions, the bot will fail to work as expected.

Limitations

The Message Backup Bot has some limitations.

  • You can only save 6 channels worth up to 400 Messages (Free) [If you want more, please, view the “Upgrades” section of the wiki]
  • Restoring may take a while
  • The “bot” tag shows next to your name.
