Account Deletion

You can simply type the command /user delete to purge your data and all infomation regarding Message Backup Bot. Please note though, it can take a maximum of 10 days for all your data to be 100% deleted due to our backup system.

Refunds for paid plans

Legally we are required to do refunds for the paid plans, but we do not fit in the physical product category. Once buying premium and refunding it, we have the rights to delete the backups you created, this way, the backup system will not be abused.

Privacy Policy

User data is hashed & encrypted with best security possible. This data is stored on our own physical server at Australia, meaning no one has access to your data.

We do not sell this data or distribute it.

  • When backing up, we create a webook in the channel, this webhook stays there after the backup. We do delete the webhook URL though, meaning we can't sending anything to it.
  • We backup:
  1. Username and discriminator
  2. Message data
  3. Files
  4. Attachments
  5. Links
  6. Emojis
  7. GIFS
  8. Stickers
  9. Thread Data (if your backing up a thread)
  10. Name of the channel

You have the rights to delete user data with the command /user delete.

Privacy Of Stored Contents

Message content is hashed with the encryption key given to the user first time on initial registration will be provided inside their direct messages so they can decrypt their data. We cannot decrypt users data, only the user can do so.
