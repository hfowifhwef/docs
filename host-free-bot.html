How to install a bot on this service?

  1. You should be familiar with Git, if not, you will have a hard time.
  2. On your git repo adress, type in your GitHub URL, install branch can should always be master. Leave User uploaded files and Auo Update to default. On your Git USERNAME, type your username
  3. For the service to function properly, type your github acess token. For github: https://github.com/settings/tokens type that URL, press “generate” and then copy and paste it in. No documentary exists for GitLab.
  4. After that, return to console and hit “start”. In a matter of minutes, your bot will start. Give your bot about 10-20 mins to load up for the first time.
