What is EncryptOS?

EncryptOS is a privacy-orientated operating system powered by SnailDOS team. This operating system gets regular updates for security and customization. This operating system is based on Arch and utilizes Linux/zen Linux kernel. 

Who is it for?

This operating system is for anyone that does not know anything about Linux and wants to stay private all the way to advanced Linux users. 

Why should I choose EncryptOS over a plain Arch installation?

EncryptOS has a large wrapper over Arch that includes features (view the “features” section) that allow any user to use EncryptOS.

What Latest EncryptOS version?

26.13.1.1 is the latest stable EncryptOS version.

Features

  • Automatic UI-enriched installer.
  • Fast, secure and free updates.
  • Inbuilt browser and (insert more info later)
  • Stable and bug-free experience.
  • Tons of customization out of the box.
  • Option to use the Zen kernel (for gaming)
  • Update notifications
  • GUI app installer supporting most popular Linux apps.
  • Welcome GUI allows updates and customization to be done all in one app.

What is planning to be added?

Development

Running a development environment is not recommended for beginners. (You can also use this to compile the OS)

 

PKGBUILD

git clone https://github.com/Encrypt-OS/PKGBUILDS
cd PKGBUILDS
sudo su
sh automate-package-build.sh
# Proceed to wait and interact with the script

Mirror

git clone https://github.com/Encrypt-OS/repo
cd PKGbUILDS
sudo su
sh automate-package-inject-mirror.sh

Host the mirror on a webserver to host the environment.

 

Compile the ISO for the OS

Using the EncryptOS mirror is required.

git clone https://github.com/Encrypt-OS/EncryptOS-ISO.git
cd EncryptOS-ISO
sudo pacman -S archiso mkinitcpio-archiso git squashfs-tools --needed
sudo ./fix_permissions.sh
sudo ./mkarchiso -v "."
