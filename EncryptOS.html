What is EncryptOS?

EncryptOS is a privacy orientated operating system powered by SnailDOS team. This operating system gets regular updates for security and customization. This operating system is based on Arch and utilizes linux/zen linux kernel. 

Who is it for?

This operating system is for anyone that does not know anything about linux and wants to stay private to advanced linux users. 

Why should I choose EncryptOS over a plain Arch installation?

EncryptOS has a large wrapper over Arch that includes features (view the “features” section) that allow any user to use EncryptOS.

What Latest EncryptOS version?

26.13.1.1 is the latest stable EncryptOS version.

Features

  • Automatic UI enriched installer.
  • Fast, secure and free updates.
  • Inbuilt browser and (insert more info later)
